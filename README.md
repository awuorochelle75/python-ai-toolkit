# Hello Flask Web App

## Overview

This is a minimal **Python Flask web application** that displays a simple "Hello, World!" message in the browser.  
It was created as part of a beginner-friendly toolkit to learn web app development using Flask and Python.

**Technologies Used:**

- Python 3.x
- Flask (micro web framework)
- Visual Studio Code (IDE)

---

## System Requirements

- **Operating System:** Windows / Mac / Linux
- **Python Version:** 3.x
- **Packages:** Flask
- **IDE / Editor:** Visual Studio Code
- **Browser:** Chrome, Edge, or any modern browser

---

### Setup Instructions

1. **Clone the repository**  
   Open your terminal and run the following command:
   ```sh
   $git clone https://github.com/awuorochelle75/python-ai-toolkit.git
   cd hello_flask

2. **Install Flask**

   ```sh
       $pip install flask

   ```

3. **Run the web app**

   ```sh
       $python app.py

   ```

4. **View in browser**

- Open http://127.0.0.1:5000/
- You should see:
  " Hello, World! Welcome to my Python web app! "

5. **Optional:Create requirements.txt for easy setup**
   ```sh
     $pip freeze > requirements.txt
   ```
